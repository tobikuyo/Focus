# Focus

Landing page for a tool that helps people to create a portfolio that showcases their work. Visitors that are interested in the service, can submit details that will be saved in a PostgreSQL database.

### Tech Stack:

-   React
-   CSS
-   NodeJS
-   Express
-   PostgreSQL
-   Jest
